**Program Details**

This program reads employee and attendance CSV files to calculate June-Dec 2024 pay. Login as "employee" or "payroll_staff" (password 12345). It splits each month into 1st pay (days 1-15, no deductions) and 2nd pay (days 16-end with SSS, PhilHealth, PagIBIG, taxes). Smartly figures work hours (8AM-5PM, minus lunch, max 8hrs/day). Clean output with bold names and organized monthly sections. Works perfectly for one person or everyone.
